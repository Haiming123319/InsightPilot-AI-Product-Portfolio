# InsightPilot V6 验证报告

更新时间：2026-07-13

## 1. V6 验证结论

V6 的核心工作流已经接入当前 Demo：

- 字段映射配置会将业务字段转换为 `date`、`department`、`expense_type`、`amount` 等标准字段。
- 清洗规则会真正影响计算，包括缺失金额处理、重复记录处理和异常值是否排除。
- 分析计划支持启用/停用、顺序调整和步骤描述编辑，并在执行前做依赖校验。
- 结果页支持“排除异常值后重新分析”等上下文追问。
- 执行日志记录任务、意图、计划、Python 执行、校验、完成和失败事件。
- 保留规则版解析器，同时新增可注入的 OpenAI 意图解析适配器。
