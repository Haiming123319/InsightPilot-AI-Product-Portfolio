# 评测计划与当前结果

## 1. 评测目标

验证 InsightPilot 是否能将用户自然语言问题稳定转成可执行任务，并通过 Python 计算和规则校验输出可追溯结果。

## 2. 当前评测对象

当前只评测方案 C 的本地规则版：

> 规则版意图解析 + Python 计算 + 规则校验

尚未完成三方案对比：

| 方案 | 状态 |
|---|---:|
| 纯 LLM | 未评测 |
| LLM + Python | 未评测 |
| LLM + Python + 校验 | 已用规则版模拟 |

## 3. 测试数据与测试集

| 文件 | 说明 |

| `data/sample_expenses_dirty.csv` | 样例费用脏数据 |
| `data/test_cases.csv` | 32 条测试问题 |

## 4. 指标

- 任务完成率
- 任务理解准确率
- 校验通过率
- 澄清率
- 平均耗时

真实 LLM 接入后再补：

- 数值准确率
- 结论一致率
- 幻觉率
- Token 用量
- API 成本

## 7. 当前结论

当前规则版 Demo 已稳定覆盖作品集主场景和扩展场景。但这个结果不能等同于真实 LLM 能力，因为意图解析仍是规则模拟。

下一步需要接入真实模型，并比较：

1. 纯 LLM
2. LLM + Python
3. LLM + Python + 校验
