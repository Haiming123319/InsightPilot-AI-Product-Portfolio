# 项目复盘

## 当前进展

已完成可演示作品集候选版：用户可以上传或加载示例费用数据，完成数据体检后输入自然语言问题，系统生成结构化意图和分析计划，经用户确认后调用 Python 完成部门汇总、同比计算、费用类型下钻、异常大额费用识别、图表生成、结论输出、公式/代码溯源、发布前校验、报告导出和批量评测。

## 产品取舍

当前仍不接入真实大模型 API，先用规则版 `llm_service` 模拟结构化输出，优先保证产品闭环、计算口径、结果溯源和校验机制稳定。这符合“先使用模拟数据完成闭环，再接入真实 API”的开发要求。
