# InsightPilot 精简 PRD

## 1. 产品定位

InsightPilot 是面向业务分析人员的可验证 AI 数据分析助手。它让用户通过自然语言完成费用数据体检、部门费用汇总、同比分析、异常大额费用识别、图表生成、结论解释和结果导出。

## 2. 核心用户

需要定期处理 Excel 费用报表，但不熟悉 Python、SQL 和复杂 BI 工具的业务分析人员。

## 3. 核心问题

- Excel 清洗和公式计算耗时，且容易出错。
- 直接让大模型回答数据问题容易编造数字。
- 业务用户不知道 AI 对数据做了什么。
- 分析结论缺少公式、代码和数据来源，难以复核。

## 4. MVP 范围

已实现：

- CSV/XLSX 单文件上传
- 示例费用数据
- 数据预览和字段体检
- 缺失值、重复值、金额格式、日期格式检测
- 自然语言问题输入
- 规则版结构化意图解析
- 分析计划生成与用户确认
- 三类分析任务：部门费用汇总、部门同比分析、异常大额费用识别
- Python 计算和 Plotly 图表
- 公式、代码、证据和校验结果展示
- Markdown 报告导出
- 批量评测与 Bad Case 日志导出

暂不实现：

- 多文件关联
- 数据库连接
- 企业权限管理
- 多 Agent
- RAG
- 模型微调
- 自动生成 PPT
- 团队协作

## 5. 用户流程

上传数据 → 数据体检 → 输入问题 → 结构化意图 → 生成分析计划 → 用户确认 → Python 执行 → 自动校验 → 图表与结论 → 查看证据 → 导出报告。

## 6. 支持的问题类型

| 类型 | 示例 | 当前状态 |
|---|---|---:|
| 描述性统计 | 各部门总费用是多少 | 已支持 |
| 对比分析 | 哪个部门 2025 年同比增长最快 | 已支持 |
| 趋势分析 | 市场部费用按月如何变化 | 已支持 |
| 下钻分析 | 市场部增长来自哪些费用类型 | 已支持 |
| 异常分析 | 哪些记录属于异常大额费用 | 已支持 |
| 错误字段 | 按客户行业分析费用变化 | 已支持澄清 |

## 7. 验证机制

- 字段合法性校验
- 数值一致性校验
- 排名校验
- 文字与数字一致性校验
- 图表与数据一致性校验
- 异常阈值一致性校验
- 因果表达限制校验

## 8. 当前评测结果

测试集：32 条问题。

| 指标 | 结果 |
|---|---:|
| 任务完成率 | 100% |
| 任务理解准确率 | 100% |
| 校验通过率 | 100% |

说明：当前评测基于规则版意图解析和样例数据，不代表真实 LLM 泛化表现。下一阶段需要接入真实模型并做三方案对比。
